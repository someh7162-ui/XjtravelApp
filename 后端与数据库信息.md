## 常用运维操作

### 重启远程后端

```bash
fuser -k 3000/tcp >/dev/null 2>&1 || true
nohup node server/index.js >server.log 2>&1 </dev/null & disown
```

建议在目录 `/home/xxxy/xjtravel-api` 下执行。

### 检查服务是否正常

```bash
curl -k https://111.20.31.227:34144/api/health
```

### 同步目的地数据到数据库

```bash
npm run sync:destinations
```

当前目的地数据源来自：`common/destination-data.js`
